# Marketing Campaign Analysis

Analisi di un dataset di marketing per ottimizzare l'allocazione del budget di campagne pubblicitarie, basata sul comportamento storico dei clienti.

## Stato del progetto
🚧 In sviluppo — Work in progress.

## Obiettivo
Rispondere alla domanda: *"Come dovrebbe un'azienda food & beverage ottimizzare l'allocazione del budget delle prossime campagne marketing, basandosi sul comportamento storico dei suoi clienti?"*

## Dataset
[Customer Personality Analysis](https://www.kaggle.com/datasets/imakash3011/customer-personality-analysis) — ~2.240 clienti con dati demografici, comportamento di spesa su 6 categorie di prodotti, utilizzo di 3 canali di vendita, e risposta a 6 campagne marketing.

## Stack tecnico
- Python 3.12
- pandas, numpy (manipolazione dati)
- matplotlib, seaborn (visualizzazione)
- Jupyter Notebook
